## Command

```bash
$ gpm runtime|rt
```

output the runtime info, useful to submit a issue

## Arguments

- **runtime|rt**: <required>

## Example

#### output the runtime info

```bash
$ gpm runtime

node:     v6.10.0
gpm:      5.3.1
arch:     x64
os:       Windows_NT 10.0.14393
platform: win32

```
